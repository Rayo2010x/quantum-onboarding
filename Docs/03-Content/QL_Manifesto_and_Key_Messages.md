## 4. Brand Voice Principles

| Principle | Do | Don't |
| :--- | :--- | :--- |
| **Authoritative** | Cite technical facts (ECDSA, NIST PQC) | Make unverifiable claims |
| **Urgent** | "The threat is imminent" | "Bitcoin will be hacked tomorrow" |
| **Grassroots** | "Independent researchers and developers" | "Our company", "Our corporation" |
| **Ethos-aligned** | Reference Satoshi, decentralization | Reference price speculation or profits |
| **Inclusive** | Explain acronyms (LN, ECDSA) | Assume deep technical knowledge |
